Predefined functions in Arrays in JavaScript.

The JavaScript method toString() converts an array to a string of (comma separated) array values: ["Banana", "Orange", "Apple", "Mango"]

The pop() method removes the last element from an array: ["Banana", "Orange", "Apple", "Mango"]

The push() method adds a new element to an array (at the end): ["Banana", "Orange", "Apple", "Mango"]

The shift() method removes the first array element and "shifts" all other elements to a lower index, returns the string that was "shifted out": ["Banana", "Orange", "Apple", "Mango","Kiwi"]

The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements, & returns the new array length: ["Orange","Apple","Kiwi"]
